# <a name="slicers-in-power-bi-tutorial"></a>ตัวแบ่งส่วนข้อมูลใน Power BI (บทช่วยสอน)
รองประฐานฝ่ายขายต้องการดูเมตริกหลายตัว สำหรับทั้งแผนก และ สำหรับผู้จัดการเขตแต่ละคน เธอสามารถสร้างรายงานแยกกันของผู้จัดการแต่ละคน หรือเธอใช้ตัวแบ่งส่วนข้อมูลได้ ตัวแบ่งส่วนข้อมูลจำกัดชุดข้อมูลให้แคบลง เพื่อแสดงในการแสดงภาพอื่น ๆ ในรายงาน ตัวแบ่งส่วนข้อมูลเป็นอีกวิธีหนึ่งของการกรอง

บทช่วยสอนนี้ใช้ [ตัวอย่างการวิเคราะห์ด้านการขายปลีก](sample-retail-analysis.md)ที่มีให้ฟรี เพื่อพาคุณทำตามขั้นตอนการสร้าง และการจัดรูปแบบตัวแบ่งส่วนข้อมูล และใช้ตัวแบ่งส่วนข้อมูลนี้กรองรายงาน ขอให้สนุกกับการค้นพบวิธีต่าง ๆ ในการจัดรูปแบบ และใช้ตัวแบ่งส่วนข้อมูล 

![ตัวแบ่งส่วนข้อมูล](media/power-bi-visualization-slicers/slicer2.gif)

## <a name="when-to-use-a-slicer"></a>เมื่อไรที่ควรใช้ตัวแบ่งส่วนข้อมูล
ตัวแบ่งส่วนข้อมูล คือตัวเลือกที่ดีที่สุดเมื่อคุณต้องการ:

* แสดงตัวกรองที่ใช้งานบ่อยหรือสำคัญ บนพื้นที่รายงานเพื่อให้ง่ายต่อการเข้าถึง
* ทำให้การดูสถานะตัวกรองปัจจุบันได้ง่ายขึ้น โดยไม่ต้องเปิดรายการดรอปดาวน์ 
* กรองตามคอลัมน์ที่ไม่จำเป็นและซ่อนอยู่ในตารางข้อมูล
* สร้างรายงานที่เฉพาะเจาะจงมากขึ้น โดยการใส่ตัวแบ่งส่วนข้อมูลข้างกับวิชวลที่สำคัญ

ตัวแบ่งส่วนข้อมูล Power BI มีข้อจำกัดดังต่อไปนี้:

- ตัวแบ่งส่วนข้อมูลไม่สนับสนุนเขตข้อมูลจากค่าที่ป้อน
- ตัวแบ่งส่วนข้อมูลไม่สามารถปักหมุดไปยังแดชบอร์ด
- ตัวแบ่งส่วนข้อมูลไม่สนับสนุดการดูรายละเอียดแนวลึก
- ตัวแบ่งส่วนข้อมูลไม่สนับสนุนตัวกรองระดับการแสดงภาพ

## <a name="create-a-slicer"></a>สร้างตัวแบ่งส่วนข้อมูล

บทช่วยสอนนี้ใช้ตัวแบ่งส่วนข้อมูลรายการ ชนิดข้อมูลตัวเลขและวันที่/เวลา สามารถมีตัวแบ่งส่วนช่วง ดู[ใช้ตัวแบ่งส่วนข้อมูลช่วงตัวเลขใน Power BI Desktop](desktop-slicer-numeric-range.md) หรือวิดีโอต่อไปนี้สำหรับข้อมูลเพิ่มเติมเกี่ยวกับ การสร้างและใช้ตัวแบ่งส่วนข้อมูลช่วง
<iframe width="560" height="315" src="https://www.youtube.com/embed/zIZPA0UrJyA" frameborder="0" allowfullscreen></iframe>

1. ในบริการของ Power BI Desktop หรือ Power BI เปิด[ตัวอย่างการวิเคราะห์ด้านการขายปลีก](sample-retail-analysis.md)ใน[มุมมองการแก้ไข](service-interact-with-a-report-in-editing-view.md) และ[เพิ่มหน้ารายงานใหม่](power-bi-report-add-page.md)
2. จากบานหน้าต่างเขตข้อมูล ภายใต้เขต เลือก**ผู้จัดการเขต**เพื่อสร้างการแสดงภาพใหม่
    
    ![แผนภูมิใหม่](media/power-bi-visualization-slicers/1-new-vis.png)
    
3. เลือกไอคอน**ตัวแบ่งส่วนข้อมูล** ![ไอคอนตัวแบ่งส่วนข้อมูล](media/power-bi-visualization-slicers/slicer-icon.png)ในบานหน้าต่างการแสดงภาพ เพื่อเปลี่ยนการแสดงภาพใหม่ให้เป็นตัวแบ่งส่วนข้อมูล 
    
    ![แปลงเป็นตัวแบ่งส่วนข้อมูล](media/power-bi-visualization-slicers/2-slicer.png)

คุณยังสามารถเลือกไอคอนตัวแบ่งส่วนข้อมูลเพื่อสร้างตัวแบ่งส่วนข้อมูลใหม่ จากนั้นเลือกหรือลากเขตข้อมูลไปยังกล่อง เขตข้อมูล เพื่อเพิ่มเขตข้อมูลลงไป

>[!TIP]
>คุณสามารถเรียงลำดับรายการใน ตัวแบ่งส่วนข้อมูล ตามค่าของข้อมูล เพื่อเรียงรายการย้อนลำดับตัวอักษร เลือกจุดไข่ปลา (...) ในมุมบนขวาของตัวแบ่งส่วนข้อมูล แล้วเลือก**เรียงลำดับตาม ผู้จัดการเขต** ค่าเริ่มต้นเป็นการเรียงตามลำดับตัวอักษร แต่สามารถสลับไปมาระหว่างเรียงจากน้อยไปหามาก และจากมากไปหาน้อยได้ 

## <a name="format-the-slicer"></a>จัดรูปแบบตัวแบ่งส่วนข้อมูล
นำการจัดรูปแบบวิชวล ไปใช้กับตัวแบ่งส่วนข้อมูลผู้จัดการเขต
1. เลือกตัวแบ่งส่วนข้อมูลไว้ก่อน จากนั้นในบานหน้าต่างการแสดงภาพ เลือกไอคอนรูปแบบ ![](media/power-bi-visualization-slicers/power-bi-paintroller.png) เพื่อแสดงตัวควบคุมการจัดรูปแบบ 
    
    ![การจัดรูปแบบ](media/power-bi-visualization-slicers/3-format.png)
    
2. คลิกลูกศรดรอปดาวน์ถัดจากแต่ละประเภท เพื่อแสดงและแก้ไขตัวเลือก 

### <a name="general-options"></a>ตัวเลือกทั่วไป
1. เลือกสีแดงภายใต้**สีเส้นขอบ** และเปลี่ยน**ความหนาของเส้นขอบ**เป็น "2" นี่เป็นการตั้งค่าสีและความหนาของ เส้นขอบของส่วนหัวและรายการ หรือการขีดเส้นใต้ เมื่อเปิดใช้งาน 
2. ภายใต้ การจัดแนว ค่าเริ่มต้นคือแนวตั้ง เป็นการสร้างตัวแบ่งส่วนข้อมูลรายการในแนวตั้ง ที่มีกล่องให้เลือกอยู่หน้าแต่ละรายการ เลือก**แนวนอน** เพื่อสร้างตัวแบ่งส่วนข้อมูล ที่จัดเรียงรายการในแนวนอน การจัดเรียงแนวนอน สามารถวางข้อความ ปุ่ม หรือไทล์ในรูปแบบต่าง ๆ กัน ขึ้นอยู่กับขนาดและรูปร่างของตัวแบ่งส่วนข้อมูล และการจัดรูปแบบรายการ 
    
    ![แนวนอน](media/power-bi-visualization-slicers/4-horizontal.png)
    
3. เปิดใช้งานเค้าโครงแบบ**ตอบสนอง** ซึ่งเปลี่ยนขนาดและการจัดเรียงของรายการในตัวแบ่งส่วนข้อมูลแนวนอน เพื่อให้สัมพันธ์กับขนาดและรูปร่างของตัวแบ่งส่วนข้อมูล เมื่อย่อให้มีขนาดเล็กมาก ตัวแบ่งส่วนข้อมูลจะกลายเป็นไอคอนรูปตัวกรอง 
    
    ![ตอบสนอง](media/power-bi-visualization-slicers/5-responsive.png)
    
    >[!NOTE]
    >การเปลี่ยนแปลงเค้าโครงแบบตอบสนอง อาจเปลี่ยนการจัดรูปแบบหัวเรื่องหรือรายการที่คุณตั้งค่าไว้ได้ 
    
4. ตั้งค่าตำแหน่งและขนาดด้วยตัวเลขที่เที่ยงตรง ด้วย**ตำแหน่ง X**, **ตำแหน่ง Y**, **ความกว้าง**และ**ความสูง** หรือย้ายและปรับขนาดตัวแบ่งส่วนข้อมูลบนพื้นที่ทำงานโดยตรง เพื่อสร้างรายการที่มีขนาดและการจัดเรียงที่แตกต่างกัน เช่น เป็นแถวของปุ่มในแนวนอน 

    ![ปุ่มแนวนอน](media/power-bi-visualization-slicers/6-buttons.png)

ดู[สร้างตัวแบ่งส่วนข้อมูลแบบตอบสนอง ที่คุณสามารถปรับขนาดใน Power BI](power-bi-slicer-filter-responsive.md) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับจัดเรียงแนวนอนและการจัดรูปแบบตอบสนอง

### <a name="selection-controls-options"></a>ตัวเลือก ควบคุมรายการเลือก
1. ตามค่าเริ่มต้น แสดงการเลือกทั้งหมดจะถูกปิดอยู่ **เปิด**เพื่อเพิ่มรายการ เลือกทั้งหมด ลงในตัวแบ่งส่วนข้อมูล เพื่อเลือกหรือไม่เลือกรายการทั้งหมด เมื่อเลือกรายการทั้งหมด การคลิกที่รายการในรายการหนึ่งจะเป็นการยกเลิกเฉพาะรายการนั้น ให้เป็นตัวกรองชนิด "ทุกอย่างยกเว้น" 
    
    ![เลือกทั้งหมด](media/power-bi-visualization-slicers/7-select-all.png)
    
2. ตามค่าเริ่มต้น เลือกรายการเดียว จะเปิด การคลิกที่แต่ละรายการ จะเป็นการเลือกรายการนั้น ถ้าต้องการเลือกหลายรายการ ให้กดแป้น CTRL ค้างไว้ขณะที่คลิก เปลี่ยน เลือกรายการเดียว ให้เป็น**ปิด** ให้สามารถเลือกหลายรายการโดยไม่ต้องกดแป้น CTRL ค้างไว้ คลิกที่แต่ละรายการอีกครั้งยกเลิกการเลือกดังกล่าว 

### <a name="header-options"></a>ตัวเลือกส่วนหัว
ตามค่าเริ่มต้น จะเปิดใช้งานส่วนหัว ซึ่งจะแสดงชื่อเขตข้อมูลที่ด้านบนของตัวแบ่งส่วนข้อมูล 
1. จัดรูปแบบข้อความส่วนหัวเพื่อทำให้**สีแบบอักษร**เป็นสีแดง **ขนาดของข้อความ** 14 pt และ**ตระกูลแบบอักษร** Arial Black 
2. ภายใต้เค้าร่าง เลือก**ด้านล่างเท่านั้น** เพื่อสร้างการขีดเส้นใต้ด้วยขนาดและสีที่คุณตั้งค่าภายใต้ตัวเลือกทั่วไป 

### <a name="item-options"></a>ตัวเลือกรายการ
1. จัดรูปแบบข้อความรายการและพื้นหลังเพื่อทำให้**สีแบบอักษร**เป็นสีดำ **พื้นหลัง**เป็นสีแดงอ่อน **ขนาดของข้อความ** 10 pt และ**ตระกูลแบบอักษร** Arial 
2. ภายใต้เค้าร่าง เลือก**กรอบ** เพื่อวาดเส้นขอบรอบ ๆ แต่ละรายการ ด้วยขนาดและสีคุณตั้งค่าภายใต้ตัวเลือกทั่วไป 
    
    ![จัดรูปแบบแล้ว](media/power-bi-visualization-slicers/8-formatted.png)
    
    >[!TIP]
    >- ด้วยการจัดแนวนอน รายการที่ไม่ได้เลือก แสดงข้อความและสีพื้นหลังตามที่กำหนดไว้ ในขณะที่รายการที่เลือกใช้ค่าเริ่มต้นของระบบ ซึ่งมักจะเป็นพื้นหลังสีดำกับข้อความสีขาว 
    >- ด้วยการจัดแนวตั้ง รายการจะแสดงสีที่ตั้งค่าเสมอ และกล่องสำหรับเลือกจะเป็นสีดำเมื่อถูกเลือก 

### <a name="other-formatting-options"></a>ตัวเลือกการจัดรูปแบบอื่น ๆ
ตัวเลือกจัดรูปแบบอื่น ๆ จะปิดใช้งานตามค่าเริ่มต้น เมื่อเปลี่ยนค่าให้เป็น**เปิด**: 
- **ชื่อเรื่อง:** เพิ่มและจัดรูปแบบชื่อเรื่อง (ซึ่งเพิ่มเติมจากส่วนหัว และตั้งค่าอิสระจากกัน) ที่ด้านบนของตัวแบ่งส่วนข้อมูล 
- **พื้นหลัง:** เพิ่มสีพื้นหลังโดยรวมของตัวแบ่งส่วนข้อมูล และตั้งค่าความโปร่งใส
- **ล็อกเป้าหมาย:** รักษารูปร่างของตัวแบ่งส่วนข้อมูลถ้าถูกปรับขนาด
- **ขอบ:** เพิ่มขอบหนา 1 พิกเซล รอบ ๆ ตัวแบ่งส่วนข้อมูล และตั้งค่าสี (ขอบตัวแบ่งส่วนข้อมูลนี้แยกต่างหาก และไม่มีผลจากการตั้งค่า เค้าร่างทั่วไป) 

## <a name="sync-and-use-the-slicer-on-other-pages"></a>ซิงค์ และใช้ตัวแบ่งส่วนข้อมูลบนหน้าอื่น ๆ
เริ่มตั้งแต่การอัปเดต Power BI เดือนกุมภาพันธ์ 2018 คุณสามารถซิงค์ตัวแบ่งส่วนข้อมูล และใช้บนหน้าใด ๆ หรือทุกหน้าในรายงาน 
1. เมื่อเลือกตัวแบ่งส่วนข้อมูลผู้จัดการเขตแล้ว บนเมนูมุมมอง เลือก**ตัวแบ่งส่วนข้อมูลการซิงค์**ใน Power BI Desktop หรือเปิดใช้งาน**บานหน้าต่างตัวแบ่งส่วนข้อมูลการซิงค์**ในบริการของ Power BI บานหน้าต่าง ตัวแบ่งส่วนข้อมูลการซิงค์ จะปรากฏขึ้น 
    
    ![ตัวแบ่งส่วนข้อมูลการซิงค์](media/power-bi-visualization-slicers/9-sync-slicers.png)
    
2. ในคอลัมน์แรก เลือก**ภาพรวม**และหน้าอื่น ๆ ที่คุณต้องการให้ตัวแบ่งส่วนข้อมูลซิงค์กัน หรือคลิก**เพิ่มทั้งหมด**เพื่อให้ตัวแบ่งส่วนข้อมูลหน้ารายงานทั้งหมดซิงค์กัน  
3. ในคอลัมน์ถัดไป เลือก**ภาพรวม**และหน้าอื่น ๆ ที่คุณต้องการให้ตัวแบ่งส่วนข้อมูลมองเห็นได้ 
4. สลับไปยังหน้า**ภาพรวม** และสังเกตลักษณะของตัวแบ่งส่วนข้อมูลและผลต่อวิชวลบนหน้าอื่น ๆ 
    - เลือกและไม่เลือกรายการต่าง ๆ แล้วสังเกตว่าวิชวลอื่น ๆ ในหน้าเปลี่ยนแปลงตาม การเลือกรายการบนหน้าใด ๆ ส่งผลต่อทุก ๆ หน้าที่มีการซิงค์กัน
    - เปลี่ยนขนาด รูปร่าง ตำแหน่ง และ/หรือจัดรูปแบบของตัวแบ่งส่วนข้อมูลบนหน้า ภาพรวม การจัดรูปแบบตัวแบ่งส่วนข้อมูลบนหน้าอื่น ๆ ที่ซิงค์กัน ไม่มีการเปลี่ยนแปลง 

### <a name="control-which-page-visuals-are-affected-by-the-slicer"></a>ควบคุมว่าวิชวลในหน้าไหน ได้รับผลจากตัวแบ่งส่วนข้อมูล
ตามค่าเริ่มต้น ตัวแบ่งส่วนข้อมูลบนหน้ารายงาน จะมีผลต่อแสดงภาพอื่น ๆ ทั้งหมดบนหน้านั้น ใช้**การปฏิสัมพันธ์กับภาพ** เพื่อป้องกันไม่ให้การแสดงภาพบางภาพเปลี่ยนแปลงตาม

1. บนหน้า**ภาพรวม** เลือกตัวแบ่งส่วนข้อมูลไว้ก่อน:
    - ใน Power BI Desktop คลิกที่เมนู รูปแบบ ภายใต้เครื่องมือวิชวล แล้วเลือก**แก้ไขการโต้ตอบ**
    - ในบริการของ Power BI ให้ดรอปดาวน์**การปฏิสัมพันธ์กับภาพ**จากแถบเมนูและเปิดใช้งาน**แก้ไขการโต้ตอบ** 
    
    ตัวควบคุมตัวกรองปรากฏขึ้นเหนือวิชวลอื่นทั้งหมดบนหน้า ![ตัวควบคุมตัวกรอง](media/power-bi-visualization-slicers/filter-controls.png)
    
2. เลือกไอคอน**ไม่มี**เหนือวิชวล เพื่อให้ตัวแบ่งส่วนข้อมูลหยุดการกรองวิชวลนั้น เลือกไอคอน**กรอง** เพื่อให้ตัวแบ่งส่วนข้อมูลเริ่มการกรองวิชวลอีกครั้ง 

ดู[การการปฏิสัมพันธ์กับวิชวลในรายงาน Power BI](service-reports-visual-interactions.md) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการแก้ไขการปฏิสัมพันธ์
