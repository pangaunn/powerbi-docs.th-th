# <a name="import-and-display-kpis-in-power-bi"></a>นำเข้า และแสดง KPI ใน Power BI
ด้วย **Power BI Desktop** คุณสามารถนำเข้าและแสดง KPI ในตาราง เมทริกซ์ และการ์ดได้

ทำตามขั้นตอนเหล่านี้ เมื่อต้องการนำเข้า และแสดง KPI

1. เริ่มต้นด้วยเวิร์กบุ๊ก Excel ที่มีแบบจำลอง Power Pivot และ KPI แบบทดสอบนี้ใช้ชื่อเวิร์กบุ๊กว่า *KPI* คุณยังสามารถ[เรียนรู้วิธีการนำเข้าเวิร์กบุ๊ก](desktop-import-excel-workbooks.md)ได้อีกด้วย  
2. เปิดรายงานและเลือกแท็บ**ผู้ทำงานยอดเยี่ยม**  เมทริกซ์*ผู้ทำงานยอดเยี่ยม*ประกอบด้วย KPI สำหรับข้อมูลตัวเลขยอดขายที่แตกต่างกันตามที่อยู่อีเมลของพนักงาน  
   
    ![](media/desktop-import-and-display-kpis/desktoppreviewfeatureon.jpg)

ขั้นตอนทั้งหมดก็มีเพียงเท่านี้ คุณสามารถใช้ KPI เพื่อเน้นแนวโน้มที่สำคัญ ความคืบหน้า หรือตัวบ่งชี้ที่สำคัญอื่นๆ
